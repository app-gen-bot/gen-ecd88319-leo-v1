### Quick Start

   ```bash
   cd apps/project-management-app/frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## ğŸ¯ Features to Test

### 1. Navigation
- Click on different projects in the sidebar
- Switch between workspaces (dropdown in header)
- Toggle sidebar on/off (menu button)

### 2. Task Views
- **List View**: Traditional task list
- **Board View**: Kanban-style board (drag and drop ready)
- **Calendar View**: See tasks by due date
- **Timeline View**: Gantt chart view

### 3. Task Management
- Click "+ Create" to add new tasks
- Click on any task to see details
- Edit task properties (assignee, priority, status)
- Add subtasks within a task
- Mark tasks as complete with checkbox

### 4. Filtering
- Use the filter dropdowns to filter by:
  - Assignee
  - Status
  - Priority
- Search for tasks using the search bar

### 5. Responsive Design
- Resize your browser to see mobile/tablet layouts
- On mobile, the sidebar becomes a drawer
- Touch-friendly interface on smaller screens

## ğŸ› Troubleshooting

### Port Already in Use
If port 3000 is already in use:
```bash
npm run dev -- -p 3001
```

### Dependencies Issues
Clear cache and reinstall:
```bash
rm -rf node_modules package-lock.json
npm install
```

### Build Errors
Check for TypeScript errors:
```bash
npx tsc --noEmit
```
