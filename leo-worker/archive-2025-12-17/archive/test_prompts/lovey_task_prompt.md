# LoveyTasks - Family Task Assigner with AI Love Notes

Create a family task management app called "LoveyTasks" that transforms mundane task assignments into loving, flattering, and sometimes funny messages using AI.

## Core Concept
When family members assign tasks to each other, the app uses AI to rewrite the task description into a loving, personalized message that maintains the task details but adds warmth, humor, and appreciation.

## Key Features

### 1. User Management
- Family account creation with multiple family members
- Each member has a profile with:
  - Name
  - Relationship role (Mom, Dad, Son, Daughter, Partner, etc.)
  - Profile picture
  - Personality preferences (formal, playful, romantic, funny)
  - Message style preferences (over-the-top romantic, dad jokes, gen-z slang, shakespearean, etc.)

### 2. Task Creation & Assignment
- Create tasks with:
  - Original/plain description
  - Assignee (family member)
  - Due date/time
  - Priority (High, Medium, Low)
  - Category (Household, Kids, Shopping, Work, Personal)
- AI automatically transforms the message based on:
  - Who's assigning (sender's personality)
  - Who's receiving (recipient's preferences)
  - Task category and urgency
  - Time of day
  - Previous interaction patterns

### 4. Task Response Features
- Recipients can:
  - Accept with a loving response (also AI-enhanced)
  - Negotiate with a counter-offer
  - Mark as complete with a celebration message
  - Add progress updates with encouraging AI responses

### 5. Family Dashboard
- Overview of all active tasks
- "Love Score" - tracks positive interactions
- Task completion streaks
- Family achievements and badges
- Weekly family MVP based on tasks and positivity

### 6. Special Features
- "Message History" - see the transformation from original to lovely
- "Surprise Me" - random extra-sweet messages for special occasions
- "Passive Aggressive Mode" (hidden feature) - for when you really need to assign dishwasher duty
- Templates for common tasks with pre-written lovely versions
- Scheduling recurring tasks with variety in messages

### 7. Notification System
- Push notifications with the lovely message
- Morning motivational message with day's tasks
- Completion celebrations
- Gentle, loving reminders for overdue tasks

## Technical Requirements
- Real-time updates when tasks are assigned/completed
- Store both original and transformed messages
- Quick AI response time (under 2 seconds)
- Mobile-responsive design
- Fun, colorful, family-friendly UI
- Emoji support throughout
- Optional dark mode for late-night task assigning

## Navigation Structure
- Login/Register
- Family Dashboard
- Create Task
- My Tasks
- Family Members
- Message History
- Settings
  - Profile Settings
  - Personality Preferences
  - Notification Settings
- Help/Tutorial

Make sure the app feels warm, fun, and brings joy to the mundane act of assigning household tasks. The AI should make every interaction feel special while maintaining the practical functionality of a task manager.
